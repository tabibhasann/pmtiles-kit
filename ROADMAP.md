# Roadmap

`pmtiles-kit` should remain a practical CLI for inspecting, validating, converting, and previewing tile archives.

## Now

- Keep PMTiles and MBTiles validation strict and well-tested.
- Keep conversion behavior explicit about metadata and tile ordering.
- Keep npm releases reproducible with provenance.

## Next

- Add richer tile statistics and zoom-level summaries.
- Add repair guidance for common archive problems.
- Add fixture coverage for more real-world metadata variants.
- Improve preview ergonomics for local debugging.

## Later

- Explore streaming conversions for very large archives.
- Add optional web-worker previews for heavier datasets.
- Add cookbook docs for MapLibre, Protomaps, and static hosting workflows.
